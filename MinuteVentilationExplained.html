Graphing the Results for Classification

Performing the equation with different input parameters gives different Minute Ventilation results. Applying a coloring scheme based on Minute Ventilation Values allows for the creations of a color classification scheme based on conditional statements about the values produced.

Large Dataset Classified

Input parameters are fixed so a narrow range of values can be expected, graphing 100+ values allows for dataset creation that more easily allows for inspecting potential spatial properties in the dataset.
